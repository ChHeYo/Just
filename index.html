Fewer components, greater obsession.
Just.css aims to provide only a few core components.
In doing so, we can contain the size of Just.css to the bare minimum
and focus on polishing the components we use frequently.

THE GRID

The grid has max-width of 1140px and is responsive when viewed on smaller devices.
Unlike conventional grid system, just.css supports 4-grid system.
